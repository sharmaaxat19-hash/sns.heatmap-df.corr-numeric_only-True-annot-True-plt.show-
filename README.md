# 🎓 Student Depression Predictor

A Machine Learning project that predicts student marks based on study-related factors such as study hours, attendance, and previous academic performance.

## 📌 Project Overview

This project uses Machine Learning algorithms to analyze student data and predict their expected marks. It helps students understand how different factors affect academic performance.

## 🚀 Features

- Predicts student marks using ML
- Data preprocessing and cleaning
- Data visualization with charts
- Model training and evaluation
- Easy-to-use Python implementation

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Jupyter Notebook / Anaconda

## 📊 Dataset
